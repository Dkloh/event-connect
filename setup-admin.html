🔐 Admin Setup

⚠️ Important: You need to be logged in and have a backend endpoint to set admin claims. This is a temporary workaround.

Find this in Firebase Console → Authentication → Users

Can't find the UID?

Follow these steps:

  1. Go to Firebase Console
  2. Select your project
  3. Click Authentication → Users
  4. Find the user by email
  5. Copy the User UID (the long ID on the right)
  6. Paste it above and click "Set as Admin"
